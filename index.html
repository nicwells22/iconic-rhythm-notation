How to Use

  • Add blocks: Select a block size from the palette, then click on the grid
  • Move blocks: Drag blocks to reposition them
  • Resize blocks: Drag the right edge of a block
  • Delete blocks: Right-click on a block
  • Playback: Use play/pause/stop to visualize the rhythm
